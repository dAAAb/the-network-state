# 為什麼是現在？

為什麼是現在？經過威斯特伐利亞民族國家近400年的發展，我們為什麼認為現狀可以改變？

首先，現狀。現代民族國家到底是什麼？就這一點而言，什麼是民族？很久以前，國家的形成是如何被 [地圖製作](https://www.jstor.org/stable/179451) 和 [印刷資本主義](https://en.wikipedia.org/wiki/Print_capitalism) 等技術革新所促成的？導致民族國家崛起的政治事件是什麼時候發生的？歷史上的替代方案又是什麼？

然後是變化。當代的催化劑是什麼？這些技術和政治發展有望改變幾個世紀的實踐？有哪些概念、圖表、計算和引文表明巨大的變化正在發生？一個網路國家甚至可能是什麼樣子的？
