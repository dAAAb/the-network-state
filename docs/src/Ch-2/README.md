# 序幕
我們的歷史是網路國家的序幕。

這**並不**明顯。正如我們[之前所描述的]()那樣，建立一個初創社會似乎是為了發展一個社群、撰寫程式碼、群眾募資土地，並最終獲得外交承認，成為一個網路國家。歷史有什麼關係呢？

簡而言之，若一家科技公司首先關於技術創新、其次是公司文化，那麼一個初創社會則相反：它以社群文化為先，技術創新其次。技術創新意味著對未來的預測，而文化創新意味著對過去的探勘。

但為什麼呢？對於像 SpaceX 這樣的科技公司，你從資料中提取時間不變的物理定律開始，這些定律告訴你原子如何碰撞與互動。對這些定律的研究讓你能做出前所未見之事，似乎證明歷史並不重要。但關鍵在於：這些物理定律以高度壓縮的形式，編碼了無數次科學實驗的成果。你是在向人類經驗學習，而不是試圖自零重新推導物理規律。為了觸控火星，我們站在巨人的肩膀上。

對於一個初創社會，我們尚無永恆的「社會數學定律」。歷史是我們最接近「人類物理學」的東西。它提供許多人類行為者如何相互碰撞與互動的描述；正確的歷史研究流程，會以壓縮形式編碼無數社會實驗的結果。你可以從人類經驗中學習，而不是從頭重新擬定社會法則。學些歷史，免得重蹈覆轍。

這是理論上的論點。觀察上的論點是：我們知道文藝復興的技術革新起於重新發現歷史；我們也知道開國元勳非常在意歷史。兩者皆是透過借鑑過去而向前邁進。因此，若你想在新的初創社會中開闢道路，歷史研究的重要性就有了可信的基礎。

邏輯上的論點也許最有說服力。想想看：使用 iPhone 比從頭建立 Apple 容易太多。作為消費者，只需點一下按鈕；但要成為生產者，就有必要理解公司是如何建立的。同理，作為既有國家的普通公民是一回事，自零建立一個國家又是另一回事。要建立新社會，首先需要理解國家如何建立，瞭解其中的流程與物流——這再次把我們帶回歷史領域。

## 為什麼歷史是至關重要的

不使用它，你就無法真正學到東西。沉浸在一種新語言中的一天勝過幾周的書本學習。嘗試用程式語言構建一個東西的一天也勝過幾周的理論學習。

同樣，我們教授的歷史是一種應用歷史：對於創業公司的未來總裁以及他們的公民、股東和員工來說，這是一個重要的工具。這是你每天都會用到的東西。為什麼？

- **歷史是你贏得爭論的方式**。想想 [1619 計劃](https://www.cato.org/commentary/1619-project-autopsy)，或大學的 [申訴研究部門](https://archive.ph/67KOh)，甚至是報紙上對一些 [不幸者](https://twitter.com/balajis/status/1196991993735462913) 的 "簡介"。你可能在挖掘加密貨幣，但這種事情背後的人在挖掘歷史。也就是說，成千上萬的人全職從事 "[冒犯性考古學](https://newcriterion.com/issues/2018/12/offense-archaeology)"，挖掘最近和遙遠的過去，尋找一些他們可以寫出來的有用事件，以進一步打擊他們的政治反對派。這是翻閱某人的舊推文的學術版本。這是武器化的歷史，作為反對派研究的歷史。僅僅依靠純粹的邏輯，你根本無法贏得與這種人的爭論；你需要事實，所以你需要歷史。

- **歷史決定了合法性**。我們用摩爾定律表示戰後電晶體密度的指數級提高。我們把同一時期藥品研發效率的指數級下降描述為埃魯姆定律--即摩爾定律的反向。也就是說，在過去的幾十年裡，美國食品和藥物管理局以某種方式主持了藥物開發成本的巨大增長，即使我們的計算機和我們對人類基因組的知識有了很大的提高。類似的現象可以在能源（能源生產停滯不前）、航空（最高速度已經達到頂峰）和建築（我們今天的建築速度比七十年前慢）中觀察到。

很明顯，即使闡述埃魯姆定律也需要詳細的歷史知識，需要了解事物過去的情況。不太明顯的是，如果我們想改變埃魯姆定律，如果我們想在物理世界中再次進行創新，我們也需要歷史。

原因是，每一個FDA背後都有一個沙利度胺，就像每一個TSA背後都有一個 911 事件，每一個 Sarbanes-Oxley 背後都有一個 [安然事件](https://www.britannica.com/event/Enron-scandal)。監管是枯燥的，但導致監管的事件卻並不枯燥。

這段歷史被用來為古老的規章制度辯護；如果你改變它們，[人們就會死！](https://www.youtube.com/watch?v=eXWhbUUE4ko&ab_channel=ReasonTV)這就是我們的歷史。因此，要使物理創新合法化，你就需要成為一個反歷史學家。只有當你比他們的支持者更瞭解監管機構的合法化歷史時，你才能建立一個卓越的替代方案：一個新的監管正規化，能夠同時解決美國監管國家的濫用和他們聲稱要防止的濫用。

- **歷史決定了道德**。宗教以歷史課為起點。你可能認為這些是編造的歷史，但它們都是歷史。遙遠的過去的故事，無論是否虛構，都描述了人類曾經的行為方式--以及他們應該如何行為。這些故事有一個寓意。

政治學說也是基於歷史課的。它們是建制派自我辯護的方式。宣傳這些歷史教訓的機制是建制派報紙，其中大多數文章並不是真正的真假，而是好壞之分。你自己試試吧。只要看一眼任何機構的頭條新聞，你就能立即領悟到它的道德教訓：××主義是壞的，我們的政府系統是好的，技術創始人是壞的，等等。如果你再深入一層，如果你問為什麼這些事情是好的或壞的，你會再次得到一個歷史教訓。因為為什麼 X-主義是壞的？好吧，讓我給你上一些歷史教育...

這些道德前提的安裝是一個零和遊戲。一個社會只能容納這麼多的道德課，因為一個大腦的道德計算能力是有限的。因此，如果 99% 的人將他們有限的道德記憶分配給 "努力工作好，任人唯賢好，嫉妒不好，慈善好 "這樣的原則，與 99% 的人將 "社會主義好，文明不好，執法不好，掠奪好 "這樣的念頭內在化，你會得到一個完全不同的社會。

- **歷史是你發展引人注目的媒體的方式**。當然，你可以編造完全虛構的故事。但即使是虛構的故事也經常有某種歷史背景。《指環王》借鑑了中世紀的歐洲，義大利西部片借鑑了狂野的西部，邦德電影受到了冷戰的啟發，等等。當然，任何政治秩序的合法化故事都會借鑑歷史。

- **歷史是加密貨幣的真正價值所在**。比特幣價值數千億美元，因為它是一個可加密驗證的歷史，即誰持有什麼 BTC。詳見[《真相機器》](https://www.amazon.com/Truth-Machine-Blockchain-Future-Everything/dp/1250114578)。

- **歷史告訴你誰在掌權**。為什麼奧威爾說，誰控制了過去，誰就控制了未來，誰控制了現在，誰就控制了過去？因為歷史教科書是由勝利者編寫的。無論是否巧妙，它們都是為了講述統治機構對其過去的敵人取得巨大勝利的故事而編撰的。在美國，大多數人知道的唯一歷史是 1776 年、1865 年、1945 年和 1965 年——革命、世界大戰和積極分子運動的罐裝歷史，它們不可避免地導致更大的政治平等的陽光高地。這與蘇聯人教給他們孩子的歷史非常相似，所有的過去都透過階級鬥爭的鏡頭來解釋，把蘇聯公民帶到今天，他們不可避免地從社會主義的中間階段走向......[共產主義](https://press.princeton.edu/books/paperback/9780691121178/everything-was-forever-until-it-was-no-more)！。中國的學童也學習類似的選擇性歷史，歐洲殖民者和日本人的（真正）錯誤被集中起來，而毛澤東的錯誤則被 [淡化](https://www.ips-journal.eu/in-focus/the-politics-of-memory/70-per-cent-good-30-per-cent-bad-2216/)。即使是任何成功的創業公司，也會講述其創始人的故事，抹去粗糙的稜角。

簡而言之，歷史教科書給了你一個 [英雄旅程](https://zh.wikipedia.org/wiki/%E8%8B%B1%E9%9B%84%E6%97%85%E7%A8%8B)，讚美其建立者在所有困難中的勝利。即使歷史處理涉及表面上的受害者，如蘇聯教科書涉及無產階級的受害情況，如果你仔細觀察，撰寫該處理的統治階級通常會證明自己是**最大的**受害者。這就是為什麼任何征服者政權的首要行為之一 [就是](https://apnews.com/article/united-nations-china-beijing-hong-kong-4fdabb78fd9f467da44ca453f3655e27) [改](https://twitter.com/OlenaHalushka/status/1533212435166920706) [寫](https://www.jstor.org/stable/369749) [教科書](https://www.amazon.com/Antifascist-Classroom-Denazification-Soviet-occupied-1945-1949/dp/0230107303)（點選這些連結），告訴你誰在負責。

- **歷史決定了你的招聘政策**。為什麼科技公司在 "多樣性 "問題上被媒體公司說教？是因為那些比科技公司的白人多 20-30 個百分點的媒體公司實際上非常關心這個問題嗎？還是因為在 2009 年印刷媒體收入崩潰之後，媒體公司努力尋找商業模式，發現某些詞能帶來流量，然後加倍努力--在這個過程中提升他們的股價並抨擊他們的競爭對手？畢竟，如果你瞭解更多的歷史，你就會知道，紐約時報公司（它引發了許多這樣的抱怨）是一個這樣的組織：控制者 Ochs-Sulzberger 家族從奴隸制中獲利，阻止婦女成為出版商，幾十年來將同性戀者排除在新聞編輯室之外，在繼承過程中只有三個直系白人男性表兄弟，最後的出版商正好是前者的兒子。

假設你是一個創始人。一旦你知道這段歷史，一旦你所有的朋友、員工和投資者都知道這段歷史，一旦你知道沒有一個所謂的勇敢的建制媒體公司會用這樣的話告訴你，你就脫離了矩陣。你已經在精神上解放了你的組織。只要你不是在經營一家以世襲裙帶關係為基礎的公司，即目前經營公司的人是從他父親的父親的父親的父親那裡繼承來的，你就比《紐約時報》公司的老闆更多元化、更民主。你不需要聽他們的說教，不需要聽他們僱傭的任何人的說教，也不需要聽他們社交圈裡的任何人的說教--這包括所有建制派記者。你現在有了道德權威，可以在法律允許的範圍內僱用你需要的人，就像 SpaceX、Shopify、Kraken 和其他公司現在做的那樣。這就是對歷史的一點了解如何恢復對你的僱用政策的控制。

- **歷史是你如何 debug 我們破碎的社會**。在工程界，許多數十億美元都花在了歷史上。不過，我們並不這樣認為。我們稱其為做事後檢查，檢視日誌檔案，也許執行一個所謂的時間旅行偵錯程式來獲得一個可重複的錯誤。一旦我們找到它，我們可能想執行撤銷操作，做一個 git revert，從備份中恢復，或者返回到以前已知的良好配置。

想想我們在說什麼：在微觀層面上，瞭解系統的詳細過去，可以讓我們找出出錯的原因。而能夠部分倒退過去，沿著不同的分支前進（透過git revert），使我們有能力修復這個錯誤。這並不意味著拋棄一切，回到山頂洞人時代的空白 git 倉庫，就像那些想 "讓時光倒流 "的傳統主義者或想結束工業化文明的無政府主義者那樣。但它確實意味著倒退一下，然後沿著不同的道路前進，因為進步有大小和方向。所有這些概念都適用於比公司規模更大的情況下的 debug--比如社會，或者國家。

你現在明白為什麼歷史是有用的了。一個單純的創業公司的創始人可以說不需要歷史，默默地將歷史研究外包給那些塑造社會法律和道德的人，就可以勉強應付。但一個初創社會的總統卻不能，因為一個新的社會涉及到相對於舊社會的道德、社會和法律創新--而這需要歷史知識。

## 為什麼歷史對初創社會來說是至關重要的 

我們已經用一些具體的例子吊起了大家的胃口，說明為什麼歷史在一般情況下是有用的。現在，我們將描述為什麼它對初創社會特別有用。

我們首先介紹一套操作上有用的工具，用於從自下而上和自上而下的角度思考過去：寫在賬本上的歷史，而不是由贏家寫的歷史。

我們用這些工具來討論一個新的利維坦的出現，即網路，一個世界上最強大的力量的競爭者，一個作為社會組織機制的上帝和國家的真正同行（和補充）。

然後，我們將把這一切彙集到本章的關鍵概念上："一誡 "的概念，這是一個有歷史基礎的社會政治創新，它將公民吸引到一個初創社會，就像一個基於技術的商業創新將客戶吸引到一個初創公司。

如果說創業公司是透過識別當今市場上的經濟問題，並以新公司的形式提出一個有技術依據的解決方案，那麼初創社會則是透過識別當今文化中的道德問題，並以新社會的形式提出一個有歷史依據的解決方案。

## 為什麼初創社會不只是關於技術？

等等，為什麼一個初創的社會必須從道德問題開始？而且為什麼解決這個道德問題的方法需要有歷史依據？難道就不能成為一個以技術為中心的社群，讓人們用方程式來解決問題嗎？我們感興趣的是火星和 [延壽](https://balajis.com/the-purpose-of-technology/)，而不是塵封已久的城市的故事！"。

保羅-約翰遜在 [這個講座](https://charlierose.com/videos/2242) 的 11 點處給出了快速的答案，他指出，早期美國的宗教殖民地比盈利性殖民地的成功率更高，因為前者有一個目的。稍微長一點的回答是，在初創社會中，你不是要求人們購買產品（這是一個經濟的、個人主義的推銷），而是加入一個社群（這是一個文化的、集體的推銷）。你在論證你的初創社會的文化比周圍的文化更好；隱含地，這意味著世界上有一些道德上的缺陷需要你來解決。歷史會發揮作用，因為你需要 (a) 寫一份關於道德缺失的研究報告，(b) 從過去的歷史中找到替代的社會安排，在這些社會安排中，道德缺失並沒有發生。技術可能是解決方案的一部分，計算也可能涉及其中，但當你深入地寫任何社會問題的時候，你會發現自己在寫這個問題的歷史。

關於具體細節，你可以跳到 [平行社會的例子]() 一節--或者你可以暫時不相信，繼續閱讀，並相信我們，這個歷史/道德/倫理角度可能是建立初創社會所缺少的成分，畢竟初創社會在現代世界還沒有完全起飛。

## 初創社會的應用歷史 
以下是本章的綱要。

1. 我們從自下而上的歷史開始。關於微觀歷史和宏觀歷史的章節彌補了一個孤立的、可重複的系統的軌跡和數百萬相互作用的人類的軌跡之間的差距。因為這些小規模和大規模的軌跡現在都可以被數字記錄和量化，這就是寫在賬本上的歷史--最終形成了比特幣的加密歷史。

2. 我們接下來討論自上而下的歷史。這是由勝利者書寫的歷史，是由泰勒-考恩（Tyler Cowen）所說的 [基業長青（BASE-Raters）](https://www.bloomberg.com/opinion/articles/2020-03-03/how-fast-will-the-new-coronavirus-spread-two-sides-of-the-debate)概念化的歷史，是為當前秩序辯護並宣稱其穩定和不可避免的歷史。這是一種政治權力與技術真理的理論。

3. 然後我們談論權力的歷史，透過確定世界上最強大的力量的三個候選人，為我們剛才描述的力量命名。上帝、國家和網路。用三個主要推動者而不是一個來框定事物，使我們能夠超越純粹的以上帝為中心的宗教來理解隱含在現代社會中的以利維坦為中心的學說。

4. 我們將此應用於權力鬥爭的歷史。透過上帝/國家/網路的視角，我們可以以不同的方式理解藍/紅和技術與媒體的衝突，作為上帝的子民、國家的子民和網路的子民之間的多面鬥爭。

5. 我們在 "如果新聞是假的，想象一下歷史 "中回顧了國家的人如何利用他們的權力來歪曲最近和遙遠的歷史，以及網路如何在新近糾正這種歪曲。

6. 在展示了歷史被扭曲的程度，從而取代了歷史弧線向美國建制派不可避免的勝利彎曲的（隱含的）歷史敘事之後，我們在《分裂》、《前沿》、《第四次轉折》、《未來是我們的過去》一節中討論了幾個關於過去和未來的替代理論。這些理論並沒有描述每個軸線上的乾淨的進步勝利，而是描述了一系列的週期、髮夾彎和映象，一組歷史軌跡遠比透過教科書和大眾媒體偷渡來的線性必然性的敘述要複雜。

為什麼？政治消費者必須從幾個黨派平臺中挑選一個，而政治創始人可以做一些不同的事情：意識形態建設。為了說明這一點，我們將展示左派和右派在歷史上是如何互換陣營的，以及任何成功的群眾運動都有革命的左派成分和統治的右派成分。

如果歷史不是預設的向一個方向彎曲，如果目前的機構可能會經歷碎片化和第四次轉折的戲劇性破壞，如果它的力量實際上來自不斷擴大的邊界而不是不斷擴大的特許權。如果歷史在某種程度上按照 "未來是我們的過去 "的論點逆向執行，如果革命階級和統治階級實際上正在轉換陣營，如果新的利維坦即網路確實在國家之上崛起，如果美國內部的衝突不能被視為政策爭端，而是神聖的戰爭，是利維坦的衝突 那麼，"基業長青 "的假設，即所有的事情都將一如既往地進行，是非常不正確的 但他們並不承認這種不正確，而是試圖利用政治力量來壓制技術真相。

創始人的反擊是加密歷史和創業社會。我們現在有一個任何機構都無法輕易破壞的歷史，即由比特幣開創並透過加密貨幣神諭擴充套件的可加密驗證的歷史。我們也有一個歷史可行性的理論，歷史是一種軌跡，而不是一種不可避免性，認為只有你付出個人努力，理想的未來才會出現。但這個理想的未來的性質究竟是什麼？

畢竟，許多群體與舊秩序有分歧，但也相互有分歧--所以一攬子解決方案不會奏效。而且很可能被抵制。這就是 "一誡 "的意義所在。

作為背景，現代人往往在道德上緘默不語，但在政治上卻喜歡傳教。他們對談論什麼是道德或不道德猶豫不決，因為他們沒有資格說什麼是正確的。然而，當涉及到政治時，這種不自信經常被對他人必須如何生活的霸道自信所取代，再加上在必要時用槍口強制執行他們的信仰的熱情。

在這個零和∞之間，在完全迴避道德討論和強加一個全面的政治學說之間，在這最後一節中，我們提出一個一：一個一的戒律。根據你對歷史的研究，建立一個有自己道德準則的新社會，並招募同意你的人加入其中。19 我們不是說你需要想出你自己的新十誡，請注意--但你確實需要一誡來建立一個新的創業社會的分化。

可能的 "一誡 "的具體例子包括："24/7網際網路不好"（導致數字安息日社會），或 "碳水化合物不好"（導致Keto Kosher社會），或 "傳統基督教好"（導致Benedict Option社會），或 "生命延長好"（導致後FDA社會）。

你可能認為這些 "一誡 "聽起來要麼是瑣碎的，要麼是不切實際的野心，但在這方面，它們與科技類似；"140個字元 "的推銷聽起來很瑣碎，"可重複使用的火箭 "的推銷似乎不切實際，但這些分別導致了 Twitter 和 SpaceX。一誡 "在另一個方面也與科技類似：它將一個創業社會集中在一個單一的道德創新上，就像科技公司是關於一個集中的技術經濟創新。

也就是說，正如我們將看到的，每一個基於 "一誡 "的創業社會都以解構建制派在某一特定領域的歷史為前提，用一個新的 "一誡 "來代替它，然後透過用它來吸引訂閱者-公民來證明 "一誡 "的社會經濟價值。例如，如果你能透過對肥胖流行病進行深入的歷史研究，吸引10萬名使用者加入你的Keto Kosher社團，然後證明他們因此而大大減輕了體重，你就證明了機構在一個關鍵領域的嚴重錯誤。這將促使他們進行改革--或者不改革，在這種情況下，你會吸引更多的公民。

一個關鍵點是，我們可以將創業公司的所有技術應用到創業社會中。融資、吸引使用者、計算流失率、做客戶支援--所有這些都有一個遊戲手冊。這只是社會即服務（Society-as-a-Service），新的SaaS。

同時，其他初創社會也同樣在透過建設進行批判，用他們自己的有歷史依據的 "一戒 "將公民從建制中抽離，從而推動其他方面的變革。最後，不同的成功變革可以被複制並合併在一起，這樣，第二代創業社會就開始以二、三或N條戒律來區別於體制。這是一個和平的、平行的、由歷史驅動的破碎社會改革的願景。
