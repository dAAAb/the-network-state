# 一誡

社群是事業第一，公司第二 
每一個新的創業社會的核心都需要有一個道德前提，一個其創始國所認同的前提，一個由更強大的國家無法刪除的數字歷史所支援的前提，一個證明其存在是對權力的正義而和平的抗議的前提。

明確地說，要建立一個與宗教一樣的道德大廈，並確定所有的實際細節，這是一個巨大的努力。我們並不建議你提出自己的十誡！但我們認為你可以提出自己的一誡。

但我們確實認為你可以想出一條告誡。一個新的道德前提。只要有一個具體的問題，歷史和科學已經讓你相信，這個機構是有缺陷的。而且你有信心在文章、影片、書籍和演講中闡述你的觀點。

這些演講類似於初創企業的 decks。但作為初創社會的創始人，你並不是一個技術企業家，告訴投資者為什麼這個新的創新更好、更快、更便宜。你是一個道德企業家，告訴潛在的未來公民一種更好的生活方式，告訴他們一件更廣泛的世界已經錯了的事情，而你的社群正在將其糾正。

透過專注於一個問題，你可以建立一個具有可控複雜性的平行社會，因為你只改變了一個文明規則。不像一個政黨，你不會在許多問題上提供一攬子交易，而人們只是淺嘗輒止地關心。透過一誡，你提供的是一個用來解決單一問題的社群，吸引的不是單一問題的選民而是解決單一問題的**推動者**。
