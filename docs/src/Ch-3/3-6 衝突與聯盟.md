# 衝突與聯盟

三極的三角關係導致了令人驚訝的複雜動態。在大蕭條時期，羅斯福的美國欣賞納粹，NYT 為他們寫了讚美詩，這在《三個新交易》和《灰姑娘眨眼》中有記載。然後，在[《莫洛托夫-裡賓特洛甫條約》](https://www.britannica.com/event/German-Soviet-Nonaggression-Pact)之後，蘇聯和納粹一起入侵波蘭，拉開了第二次世界大戰的序幕，蘇聯在納粹與英美的戰鬥中袖手旁觀，而與美國結盟的英國則 [認真考慮轟炸蘇聯](https://warontherocks.com/2015/07/warchives-that-time-britain-and-france-almost-bombed-the-soviet-union/)。後來，蘇聯和納粹在巴巴羅薩期間相互爭鬥。然後，美國和蘇聯聯手打擊納粹。最後，美國和蘇聯在冷戰期間將德國分割開來，相互爭鬥。這就是為什麼奧威爾在《1984》中寫到「大洋洲一直在與歐亞大陸交戰」——因為國家之間的聯盟一直在轉換。

在網路而不是國家的情況下，聯盟甚至更加不穩定，有幾個聯盟同時存在。

## 一極對抗另一極 

NYT 對 CCP。這是顯而易見的，修昔底德的陷阱，許多人已經預測到美國和中國之間的大國衝突。但這裡有一個微妙的問題。許多普通中國人不希望發生這樣的衝突，許多美國人也不希望，但那些投資於雙方的帝國野心的人——報紙訂戶和黨員——卻喜歡這樣。網路正在推動各州的戰爭。

NYT 與 BTC。這是另一個明顯的問題，美國的監管國家（NYT 是其上游）反對去中心化的網路。我們看到了這種推動力，比如失敗的 2021 年眾議院法案和「關注科技」的信件。請注意後者的簽名者的人口統計：幾乎全是西方白人，他們抱怨美國機構失去對全球金融系統的根本控制。他們對美元的熱情是否會得到受到通貨膨脹打擊的美國人的認同--或者海外人士的認同，這一點很值得懷疑。這場衝突是美國機構與全球網際網路的衝突。

CCP 對 BTC。這是另一個明顯的衝突。多年來，中國共產黨已經多次「禁止」比特幣，但這些禁令的嚴厲程度有了實質性的提高。最近的一次行動只差沒有查封了。
