# NYT, CCP, BTC

今天的世界正在變成三極。它是 NYT 對 CCP 對 BTC。這就是美國建制派對中國共產黨對全球網際網路。

這三極中的每一極都有一個線上資訊源：紙張（紐約時報）、黨派（CCP）或協議（BTC）。每一個都有一個圍繞該資訊源的數字經濟：美元經濟，數字人民幣，或網路的加密經濟。每一個極點本身就是一個網路，它站在國家之外；NYT 的網路給美國國家指明方向，CCP 的網路領導中國國家，而 BTC 的網路則站在所有國家之外。而每個網路都有一個治理它的意識形態。

- [覺醒資本](https://www.nytimes.com/2018/02/28/opinion/corporate-america-activism.html) 是美國統治階級的意識形態，由美國的統治報紙《紐約時報》闡述。它是使分散的審查制度、取消文化和美國帝國成為可能的資本主義。它是 [無人機打擊](https://twitter.com/SirajAHashmi/status/1534537420980596736) 式的民主。

- 共產主義資本是中國共產黨的意識形態。它是由中國共產黨領導的中央集權控制的資本主義，正如 [這裡](https://twitter.com/balajis/status/1320016984826810368) 所總結的。列寧主義、儒家思想、資本主義和民族主義。

- 加密資本是比特幣和 Web3 背後的國際意識形態。它由，無國家、無公司的資本主義，對去中心化審查的抵抗，和中立的國際法所組成。它是美國和中國**內部**的第二極，是被各自國內政權的反對者所圍繞的一極。

雖然這些意識形態的表面方面可能會隨著情況的變化而變化，但我們聲稱這些是唯一擁有十億人口規模和技術才能的聯盟，可以在已經開始的全面數字鬥爭中作為獨立的權力中心生存。他們確實有內部分歧，我們會提到這一點，但就目前而言，從公司到國家，再到國家**內部的**異見派別，每一個團體都必須在這些極之間遊走，即數字世界的三極所組成的三角。
