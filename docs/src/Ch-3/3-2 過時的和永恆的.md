# 過時的和永恆的
在我們進一步討論之前，讓我們注意：就其性質而言，任何關於當前事件的文章都有可能變得過時。

一種可能是，甚至很有可能，美國當權者再次脫胎換骨，淡化象徵性，強調對國家的忠誠，就像他們一夜之間從 "全球反恐戰爭" 過渡到在你的推特上發生內戰一樣。

另一種可能是，雖然這種可能性更小，[喬治-索羅斯](https://www.project-syndicate.org/onpoint/davos-address-open-society-against-russia-china-by-george-soros-2022-05)、[彼得-澤漢](https://twitter.com/PeterZeihan/status/1444295095738585095)、[戈登-張](https://www.amazon.com/Coming-Collapse-China-Gordon-Chang/dp/0812977564) 和 [羅傑-加塞德](https://www.amazon.com/China-Coup-Great-Leap-Freedom/dp/0520380975) 最終被證明是正確的，習近平在 2022 年的黨代會上被從中國共產黨的頭把交椅上趕下來，和/或中國共產黨重新回到 "隱藏實力，等待時機" 的狀態中。

還有一種可能是，儘管也較小，發生了一些致命的缺陷，例如數學上的突破，或量子計算機，導致比特幣協議不可挽回的失敗。

那麼，既然有這麼多超出情況的事件，為什麼還要用一章來討論 NYT/CCP/BTC 模式呢？有三個原因。

首先，我們需要**一些**關於世界現狀的建模，即使是不完美的，以引導它走向我們想要的方向。即使它是錯誤的，或者在某些細節上是錯誤的，它可能是有用的錯誤，因為那些更新將顯示了我們的錯誤所在。我們花精力去描述一個具體的世界三極模式，因為許多人仍然認為它是單極或兩極，正如一位記者和印度外交部長 S.Jaishankar 之間的 [有趣互動](https://economictimes.indiatimes.com/news/india/india-not-sitting-on-fence-entitled-to-have-its-own-side-s-jaishankar-on-russia-ukraine-war/articleshow/91993466.cms) 所說明的那樣。

其次，即使真的發生了重大變化，美帝國的衰落、中國的崛起和加密貨幣的崛起仍然是涉及數億人的基本趨勢，需要巨大的力量來阻止。我們會注意到。我們將在 [之後]() 考慮這種巨大力量的幾個候選人。

其三，當前的時刻有一些方面根本沒有過時，而是反覆出現。也就是說，類似的三極配置以前也發生過。但首先讓我們確定它今天是如何產生的。
