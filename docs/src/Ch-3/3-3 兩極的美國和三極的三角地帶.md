# 兩極的美國和三極的三角地帶

1990 年，當蘇聯明顯分崩離析時，查爾斯-克勞特哈默寫了一篇有影響力的文章，名為 [單極時刻](https://www.jstor.org/stable/20044692) 。這篇文章指出，隨著冷戰的結束，美國是地球上唯一的主導力量，並將持續大約一代人的時間，之後，"多極化將及時到來。"這一論點得到了很好的支援：單極化在20世紀90年代是真實的，在2000年代大部分是真實的，在 2010 年代，隨著亞洲的崛起、技術和美國的兩極化，單極化的真實性大大降低，在 2020 年代不再真實了。

從 2022 年起，我們不再有一個單極世界了。它也不再是模稜兩可的多極，有不確定數量的權力中心。相反，我們有一個兩極的美國和一個三極的三角。而我們可以將這些極地形象化為以下幾點。
