# 臣服、同情、主權

每一極都透過呼籲一個對社會有用的概念來使自己合法化，並將其推向極端，作為譴責其相反極端的一部分。

CCP 是最明顯的：你必須服從。他們是 CCP，他們很強大，所以你必須低頭。這非常簡單明瞭，容易理解，儘管這隻在中國和中國網際網路內對他們真正有效。

NYT 的觀點稍顯微妙：他們要求你必須**表示同情**。畢竟，你不是白人、男性、異性戀、陰陽人、殘疾者、富有者，或者是一個不斷增加的特權類別的成員--所以你是在**某些**方面的壓迫者？因此你**有權力**，你必須表示同情，並向你表面上被壓迫的人低頭。這是一種左手版本的服從意識形態。它可以以賦予他們權力的名義讓任何人低頭，因為世界上 99.99% 的人至少在某些方面是"壓迫者"。這一極在英文網際網路上最強，在中國網際網路上最弱，而在這一極之外則是中等強度。

BTC 一極則與這兩種情況相反。它要求你必須擁有**主權**。這意味著你不能向 CCP 屈服，也不能像 NYT 要求的那樣割腕自殺，而是要抬頭挺胸。你在本地持有你的私鑰，你不相信中心化的公司或政府，你是自給自足的，你生活在電網之外。這一極在全球網際網路上很強大，儘管它正面臨著 CCP 和 NYT 的反擊。

## 極端和反極端是不可取的 

這裡的微妙之處在於，這些極中的每一極都有一個真理的因素。你不希望有一個 CCP 的社會，每個人都沒有辦法，只能服從，因為這很容易成為一個現在的數字極權主義。另一方面，你也不希望出現一個沒有人服從任何人的社會，因為那看起來就像舊金山，人們可以衝進沃爾格林超市，偷走一切。

你不希望在 NYT 管理的社會中，每個人都無計可施，只能同情 [當前的事情](https://knowyourmeme.com/memes/i-support-the-current-thing)，因為這導致了馬特-伊格萊西亞斯（Matt Yglesias）所說的 [大覺醒](https://www.vox.com/2019/3/22/18259865/great-awokening-white-liberals-race-polling-trump-2020)：情緒化和非理性的崩潰，讓美國著火，並繼續攪亂美國社會。然而，你也不希望出現**沒有人**同情的社會，因為那看起來像 [90 年代俄羅斯](https://www.theguardian.com/news/2018/mar/23/how-organised-crime-took-over-russia-vory-super-mafia) 的《俠盜獵車手》環境，即低信任度的後共產主義社會，任何合作努力都被視為騙局。

最後，也許是最不明顯的，你不想要這樣一個社會，每個人都**必須**擁有主權，因為如果達到其非理性的極限，這意味著從地下抽出自己的水，種植自己的食物，不相信任何供應商或自己以外的人，並普遍結束使資本主義執行的勞動分工。極端的自給自足可能聽起來很浪漫，但在沒有機器人突破的情況下，真正的離網是生活水平急劇倒退的秘訣。相反，你當然不希望一個沒有人有可能成為主權者的社會，因為這讓我們都受到 CCP 已經推出的、[NYT](https://www.nytimes.com/2020/10/13/magazine/free-speech.html) [希望](https://www.nytimes.com/2019/10/04/opinion/sunday/free-speech-social-media-violence.html) 推出的不太原始的數字極權主義的影響。

## 再中心化的中心 

有人可能會說——我也同意——雖然這三個極點及其相反的三個極端都很糟糕，但它們並不都是**同樣**糟糕的，而且你不一定需要處於中心位置。例如，我個人會比我們目前的文化更接近主權極，並嘗試開發技術來實現這一點。

然而，我們應該認識到，不同的筆觸會適合不同的人。而不是試圖把偏好強加給每個人，我們真正想要的是介於這三個不理想的極點之間的**各種**點：不同群體的不同融合。

我們在本書中我們提出的建構——最終將成為網路國家的初創社會——理想地結合了所有三個方面。例如，它確實有一個明確的創始人來提供方向，但它確保每個公民都有權利在他們選擇時自由離開，硬幣持有者也有發言權，以及其他一些數字檢查和平衡。這個概念是再中心化的基礎，我們將在後面深入討論這個想法。
