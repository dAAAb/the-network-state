# 一張圖中的網路國家

![](https://book.thenetworkstate.com/assets/dashboard-1729314.png)

這張圖片能幫助你理解。上面的儀錶板顯示了一個百萬人口的網路國家在地圖上的樣子。具體來說，它描繪了一個擁有 170 萬人口、年收入超過 1570 億美元、佔地面積為 1.36 億平方公尺的網路國家。

我們注意到的第一件事是，網路國家不像民族國家那樣在物理上集中，也不像城市國家那樣在規模上有限。它在地理上是分散的，透過網際網路連結。

我們看到的第二件事是，你可以從你的電腦上可行地啟動這種國家。也就是說，就像 Facebook 從一個人的筆記型電腦發展起來一樣，一個擁有全球實體領土群島的百萬人網路國家可以從一個人的初創社會開始，如這個 gif 圖所示：[thenetworkstate.com/networkstate.gif](https://thenetworkstate.com/networkstate.gif)。

我們看到的第三件事是即時人口普查對網路國家的核心作用。所示的儀錶板結合了代幣、公司與國家的概念，將一個社會的重點放在人口、年收入與不動產足跡的成長上。

持續的成長是一種持續的公民投票，是由留在裡面的人與外面申請加入的人所進行的信任投票。粗略地說，一個成功的網路國家是吸引結盟移民的國家，而一個不成功的網路國家則是流失他們的國家。

這並不意味著每個網路國家必須發展到無限大，也不意味著所有國家都需要接受同一種人，而是意味著網路國家的社群作為整體，專注於建立人們願意加入的令人欽佩的社會。不同的國家將專注於不同的指標；想像一個網路國家的前提是提升其公民的整體[預期壽命](https://twitter.com/balajis/status/1287395551487524864)，或者一個旨在證明正確改變所有人的[收入分配](https://www.gapminder.org/fw/income-mountains/)的國家。你將得到你所測量的東西。
