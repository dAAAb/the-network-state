# 一句話描述網絡國家

用一句非正式的話來說：

> 網絡國家是一個高度一致的線上社群，具備集體行動的能力，能在世界各地集資拓展領土，並最終獲得既有國家的外交承認。

當我們想到一個民族國家時，我們立刻會想到土地；但當我們想到一個網絡國家時，應該立刻想到的是思想。也就是說，如果民族國家體系是從全球地圖出發，將每一塊土地分配給一個國家，那麼網絡國家體系則是從世界上超過七十億人開始，將每個人的思想吸引進入一個或多個網絡之中。

這裡有一個更完整的定義，它擴展了這個概念並預先涵蓋了許多邊緣情況：

> 網絡國家是一個具備道德創新、某種民族意識、公認創辦人、集體行動能力、個人層面的文明程度、一種整合的加密貨幣、受社會智慧合約約束的共識型政府、群眾募資購得的實體領土群島、虛擬資本與鏈上人口普查，並能以具體證據證明其擁有足夠規模的人口、收入與不動產，以取得一定程度外交承認的社群。

好吧，這聽起來像是一個口號！它之所以冗長，是因為許多網際網路現象雖然具備部分網絡國家的特徵，但並非全部。例如，比特幣、Facebook 和 DAO 都不是網絡國家，因為它們缺乏某些關鍵特質——例如外交承認——而這正是我們認為下一代民族國家的核心。

（如果你想深入了解，我們會在第五章對定義中的每個部分進行更詳細的說明。不過，若你從頭讀到尾，整體理解會更加清晰。值得一提的是，民族國家的技術定義也具有類似的[多因果關係](https://archive.ph/weQQ0)，因為它需要排除我們通常不會視為國家的情況，例如無國籍民族。）
