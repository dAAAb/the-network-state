# 一千字中的網路國家

技術使我們能夠創辦新公司、新社群和新貨幣。但我們能不能用它來創造新的城市，甚至新的國家？一個關鍵的概念是先有雲，後有地——但不是永遠沒有地——從一個線上社群開始，然後將其具體化到實體世界。我們透過七個步驟達到這個目的。

1. 創立一個初創社會。這是一個簡單的線上社群，對更大的事物有渴望。任何人都可以建立一個，就像任何人都可以創立公司或加密貨幣一樣。而創辦人的正當性來自於人們是否選擇追隨他們。

2. 把它組織成一個能夠採取集體行動的團體。如果有一個足夠專注的線上社群，下一步就是將其組織成一個網路聯盟。與社群網路不同，網路聯盟有一個目的：它協調其成員，使他們共同受益。而且，與傳統的工會不同，網路工會的建立並不只是為了反對某個特定的公司，所以它可以採取各種不同的集體行動。聯盟化是一個關鍵步驟，因為它把一個原本無效的網路社群，轉化為一群為共同事業而工作的人。

3. 建立線下的信任和線上的加密經濟。開始在實體世界中舉行面對面聚會，規模與時間逐步拉長，同時使用加密貨幣建立一個內部經濟。

4. 群眾募資實體節點。一旦建立足夠的信任並累積資金，就開始群眾募資公寓、房屋，甚至城鎮，將數位公民帶入真實的共同生活社群之實體世界。

5. 以數位方式連結實體社群。將這些實體節點連結起來，形成一個網路群島——一組以數位連線串起、分佈在世界各地的實體領土。網路群島的節點範圍從一個人的公寓到任意規模的社群。持有 Web3 加密護照即可取得實體出入權，並以混合實境技術無縫連結線上與線下世界。

6. 進行鏈上人口普查。隨著社會規模擴大，進行可加密稽核的人口普查，以證明你的人口、收入與不動產足跡的成長規模。這是初創社會在面對懷疑時證明其吸引力的方法。

7. 獲得外交承認。一個具有足夠規模的初創社會，最終應能透過談判獲得至少一個先存政府的外交承認，並從那裡逐步增加主權，慢慢成為一個真正的網路國家。

關鍵的想法是要從雲端填充土地，而且要在地球上這麼做。與意識形態上不一致、地理上集中的傳統國家不同，它把數百萬的爭端者集中在一地；而網路國家在意識形態上一致，但地理上是分散的。人們分佈在世界各地，形成大小不一的聚落，但他們的心都在同一個地方。

隨著初創社會的人口與經濟成長與傳統國家相當，擁有數百萬公民與數十億收入，它最終應能獲得現有主權國家的承認——最終是聯合國——就像比特幣如今已成為一種真正的[國家法定貨幣](https://twitter.com/nayibbukele/status/1402507224916836352)。
