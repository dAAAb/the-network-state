# 前言

你是那種略過開頭就想知道是否要讀完的人嗎？你很幸運。

我們已經為初創社會與網路國家背後的概念，分別準備了一句話、一張圖、一千字與一篇文章的摘要。若你不耐煩，就點選這些連結；當然，為了完整體驗，你也可以一頁頁閱讀。

說到頁面，這本書的每一節都在「線上」，可作為單獨網頁分享。例如本節的 URL 是 thenetworkstate.com/preamble。這讓你能直接連到書中的任一重點進行討論。此外，與那種典型的「凝固在時間裡」的書不同，請把它視作持續更新的動態圖書應用程式。你可以線上上看到最新版本，或依 thenetworkstate.com/kindle.gif 的說明，把最新版本送到你的 Kindle。

在閱讀時，請把本書視作工具箱，而非宣言。你不需要同意它的全部內容，也能從中獲得一些啟發。為此，我們以「模組」形式組織：第一章是總覽；第二、三、四章提出一組分析，導向對不久未來的預測——美國式無政府與中國式控制的問題；第五章則介紹我們提出的解法：在不自由的世界中維持自由價值的途徑——初創社會與網路國家。

如果你是美國建制派，或中共黨員，你可能完全不同意我們的問題陳述；如果你是正統的比特幣至上主義者，你也可能不同意我們提出的解法每個面向；如果你來自其他學派，你或許只同意我們提出的問題或解方的一部分。儘管如此，我們相信「網路國家」這個想法足夠靈活，你可以客製化，讓它成為你的版本。

但究竟什麼**是**網路國家？
