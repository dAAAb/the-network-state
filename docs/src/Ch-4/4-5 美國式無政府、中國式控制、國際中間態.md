# 美國式無政府、中國式控制、國際中間態

在這裡，我們對一個科幻場景做了更詳細的介紹，在這個場景中，美國陷入了混亂的第二次美國內戰，中國共產黨則以相反的極端方式回應，即建立一個完全的監控國家，以將財富困在其數字人民幣的網路中，而世界其他地方——如果我們幸運的話——則建立了一個穩定的作為替代選擇的初創社會，和平地拒絕這些極端的做法。

明確地說，你不需要相信這個場景來建立初創社會和網路國家。但這是一個未來的心理模型，我們提出這個模型的原因與雷·達利奧提出的美國秩序如何落入外部競爭者之手（有點委婉），以及彼得-特欽提出的美國如何陷入內部混亂（不那麼委婉）的模型一樣。

## 美國的無政府狀態 
### 繁榮、暴政、還是無政府狀態？
進步派的觀點是，西方正在變得更加自由、平等和繁榮。歇斯底里派的觀點是，我們實際上正處於暴政的萌芽階段，無論是法西斯主義還是清醒主義。理論上不充分的是第三種可能性：即，至少在美國，民主黨和共和黨之間沒有結果的權力鬥爭意味著美國正走向 [無政府狀態](https://twitter.com/balajis/status/1484814748734550018)。
