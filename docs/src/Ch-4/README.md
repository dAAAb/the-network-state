# 可能的未來

這不是關於**那個**未來，而是關於**可能的**未來。

為什麼？因為因果關係的存在。因為我們可以進行受控實驗。因為人類的行動可以影響結果。因為我們不是相信烏托邦結果的歷史必然性的共產主義者，而是相信受實際限制的個人主動性的技術主義者。

前兩章是關於這些限制，關於過去和現在。它們引導我們討論幾個可能的未來，然後挑選出一個軌跡來關注——在這個軌跡中，我們實現了許多初創社會，使一些國家在外交上被認可為網路國家，並透過一個再中心化的中心重建高度信任度的社會。

不過，在我們開始之前，有一些注意事項。

說到過去，每段歷史都不可避免地只是一個故事。也就是說，任何關於過去的故事都必然是經過刪節、縮寫、編輯和差異化。你不可能用其他方式來表達 5000 年的書面記錄。而我們的 "作為軌跡的歷史" 一節的故事也不例外：它就像從每個企業家的 deck 開始時的 ["為什麼是現在"](https://archive.ph/fJqca#selection-135.128-143.128) 的幻燈片，是導致網路狀態可行性的特定事件的實際歷史。但我們引用了我們的參考資料，所以你可以檢查我們的事實。

關於現在的話題，我們關於 "三極時刻" 的章節是本書中最有可能過時的部分。有意如此，因為我們努力將大多數提及當前或過去的事件的內容移到這一節。因此，請將這一章看作是 2022 年中期的世界觀；就像 [卡爾曼過濾器](chrome-extension://bocbaocobfecmglnmeaeppambideimao/pdf/viewer.html?file=https%3A%2F%2Fstanford.edu%2Fclass%2Fee363%2Flectures%2Fkf.pdf) 一樣，我們保留納入 [新的資訊](https://archive.org/details/kalman_filter_excerpt) 以更新它的權利。

現在是關於未來的話題。正如你將看到的，我們確實相信一個重新集中的務實的網路狀態的中心可以出現，並描述了幾種可能發生的情況。但我們的預測只是設想，而且我們始終牢記波動性、反身性、競爭性曲線，以及隨之而來的對可預測性的限制。

首先，**波動性**正在上升，因為 [網際網路增加了變數]()。社群媒體是社會波動性（走紅或被取消），而加密貨幣是金融波動性（上月球或被打倒）。波動性使正確的預測更加困難，但為那些預測正確的人提供了上升空間。而且，波動性對叛亂者有利，對在位者不利，因為前者只需要走一次運，而後者則需要一直保持幸運。受高波動影響的不再僅僅是個人，因為整個國家可以在一夜之間崛起和衰落。因此，在高波動性環境中，只有 [貝索斯式的不變數](https://www.zdnet.com/article/jeff-bezos-business-advice-think-about-whats-not-going-to-change/) 保持不變。所有其他的觀察都應該被視為暫定的——它們是真實的，直到它們突然變得不真實。

反身性是 [索羅斯的術語](https://www.investopedia.com/terms/r/reflexivity.asp)，指的是參與者對某種情況的理解和他們所參與的情況之間的反饋迴圈。在由人類組成的系統中，把東西放到世界上就會產生反應，然後對這種反應產生反應，如此反覆，往往導致正負反饋迴圈，而不是教科書式的趨向平衡。因此，在收集這類系統的資料時，更不用說預測了，人們必須牢記，人們自己會對預測做出反應，有時會使預測成真。在社會科學中，與物理科學不同，資料集中的每一行都代表著一個有自己思想的人。

**競爭曲線**的概念指的是，目前有許多同時進行的技術政治運動在競爭，不同的現象在幾年、幾個月、甚至幾天的時間裡從零上升到影響數百萬人。例如，如果你看一下這個 [關於人們如何認識他們的配偶的圖表](https://news.stanford.edu/2019/08/21/online-dating-popular-way-u-s-couples-meet/) ，你可以看到幾個不同的曲線隨著不同的文化運動 "上線" 而上升和下降，直到網際網路剛剛主導了一切。另一個例子是 [社交網路的市場份額隨著時間的推移](https://www.researchgate.net/figure/Social-Media-Sites-unique-visitor-count-throughout-the-time-Source-Inc10b_fig1_259745791)；第三個例子是雷-達利奧的 [國家興衰圖](https://archive.org/details/dalio-chart)。

問題是，在一個複雜的多因素過程中，你可以確定參與者，但不一定能確定結果。把這一點應用到我們的情景分析中，我們有一些趨勢是協同的，還有一些是對立的。例如，許多趨勢線指向美國力量的減弱，但至少有一條指向另一個方向：西方願意為國內和國外的衝突將其科技巨頭武器化。這讓美國的主導地位又多了幾年、十年，還是比這更多？我們可以確定曲線，但不一定是哪條曲線勝出。
